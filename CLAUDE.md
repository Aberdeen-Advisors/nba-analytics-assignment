# CLAUDE.md — NBA Analytics Assignment

## Project Overview

This is the Aberdeen Advisors Intern Program NBA Analytics Challenge. Interns play the role of an analytics consultant for the fictional **Aberdeen Angels** NBA team, tasked with finding the best-value player to sign before the trade deadline given a $30M budget.

## Repository Contents

| File | Purpose |
|------|---------|
| `NBA Analytics Packet.html` | Full assignment guide — 8 phases, data dictionary, KPI guide, glossary |
| `Mock-Dashboard-Reference.html` | Reference mockup of the 3-tab Power BI dashboard interns will build |
| `nba_data_raw.xlsx` | Raw NBA player dataset (430+ players, 2024–25 season) with intentional data quality issues |

## Key Concepts

- **Bang for Buck** (`win_shares_per_10m`): the primary KPI — Win Shares divided by salary per $10M. The most important metric for comparing player value.
- **GM Filters**: salary < $30M, contract year 3–5, games played ≥ 40.
- The dataset contains **4 intentional data quality issues** that interns must find, log, and fix in Phase 3.

## Assignment Phases

1. Understand the Goal
2. Review the Dataset
3. Clean the Data
4. Pick Your KPIs
5. Find Your Top 3
6. Build the Dashboard (Power BI)
7. Write Your Insights
8. Present to the Partners
