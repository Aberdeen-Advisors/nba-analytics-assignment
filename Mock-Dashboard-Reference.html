📐 Power BI Setup for This Tab

Tab 2 compares only your 3 selected players. The easiest way to do this in Power BI is to create a separate table that contains just those 3 players' stats — either by manually entering the numbers or by using a DAX filtered table.

Then build your charts using that table instead of the full player dataset. This keeps your comparison clean and prevents other players from showing up.

DAX hint for a filtered table: SelectedPlayers = FILTER(Players, Players[player_name] IN {"Player A", "Player B", "Player C"}) — replace with your actual player names.
